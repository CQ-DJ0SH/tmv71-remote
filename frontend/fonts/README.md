# Bundled fonts

All fonts below are self-hosted (no external Google Fonts / CDN requests) and
licensed under the **SIL Open Font License, Version 1.1**, which permits
bundling and redistribution — including in commercial and open-source projects.
The full license text for each family is in `licenses/`.

| Font | Usage | Source | License |
|------|-------|--------|---------|
| Saira | UI text | Omnibus-Type (Google Fonts) | [OFL](licenses/OFL-Saira.txt) |
| Saira Condensed | Display / headings | Omnibus-Type (Google Fonts) | [OFL](licenses/OFL-SairaCondensed.txt) |
| IBM Plex Mono | Monospace / readouts | IBM (Google Fonts) | [OFL](licenses/OFL-IBMPlexMono.txt) |
| DSEG7 Classic | (bundled, optional 7-segment display) | keshikan | [OFL](licenses/OFL-DSEG.txt) |

The OFL requires that the license is distributed with the fonts and that the
fonts are not sold on their own. Reserved Font Names must not be used for
modified versions.

## Neuropol (app title) — note

`Neuropol.woff` (Ray Larabie / Typodermic) is **not** under the OFL. It is a
freeware face whose redistribution terms are not clearly documented. It is fine
for private/LAN use, but before publishing this project (even for free) verify
its license or replace it with an OFL face such as Orbitron.
